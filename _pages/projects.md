## TMO: Temporal Multi-Omics

Developed a deep learning framework using asymmetric cross-modal attention to learn signed, cell-state-conditional regulatory lags ($\Delta\tau$) from paired single-cell ATAC+RNA data. Achieved Lag Concordance Scores (LCS) of 0.988–0.999, outperforming a symmetric baseline ($<0.11$). Validated on four 10x Multiome datasets with ChIP-seq and Perturb-seq. Open-source Python package `tmopy`.

[GitHub Repository](https://github.com/phabel-LD/tmo) | [bioRxiv](https://doi.org/10.64898/2026.06.08.730880)
