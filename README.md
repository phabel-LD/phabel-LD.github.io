# Phabel A. Lopez-Delgado – Academic Website

This is the source code for my personal academic website, hosted at [phabel-LD.github.io](https://phabel-LD.github.io). It is built with [Jekyll](https://jekyllrb.com/) using the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template.

---

## About Me

I am a computational biologist and data scientist with a Licenciatura in Genomic Sciences from the Universidad Nacional Autónoma de México (UNAM). I am an author of 5 preprints (4 clinical, 1 computational) and the developer of **TMO**, a transformer-based framework for learning cell-state-dependent regulatory lags from single-cell multi-omic data.

I am passionate about interdisciplinary research, applying deep learning and statistical methods to solve problems in genomics, clinical research, and sports science.

## Running Locally

To preview the site locally:

```bash
# Install dependencies
bundle install

# Serve the site
bundle exec jekyll serve

# Open in your browser
http://localhost:4000
```
